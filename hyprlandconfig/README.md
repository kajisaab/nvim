# HyprFlux/Hyprland Configuration Backup

This directory contains a complete backup of your HyprFlux/Hyprland configuration, ready for deployment on a fresh Arch Linux installation.

## What's Included

### Features Extracted:
1. **Waybar Customization**
   - Dynamic workspace display
   - Battery percentage indicator
   - Audio volume percentage
   - Network status display
   - Custom modules and styling

2. **Wallpaper Automation**
   - Automatic wallpaper change every 30 minutes (configurable)
   - Manual wallpaper selection scripts
   - Random wallpaper switching

3. **Keybindings**
   - SUPER + Return → Kitty terminal
   - SUPER + D → Rofi launcher
   - SUPER + F → File manager
   - And many more (see UserKeybinds.conf)

4. **Terminal Configuration**
   - Kitty with JetBrains Mono Nerd Font
   - Custom color scheme
   - 80% opacity background

5. **Rofi Configuration**
   - Custom themes
   - Application launcher setup
   - Integration with HyprFlux scripts

6. **Fonts and Icons**
   - JetBrains Mono Nerd Font
   - Papirus Dark icon theme

7. **Display Settings**
   - Multi-monitor configuration
   - Generated by nwg-displays

8. **LazyVim/NvChad Configuration**
   - Complete Neovim setup
   - Custom plugins and configurations

## Quick Start

### 1. Extract Configurations (if not already done)
```bash
cd ~/Documents/hyprlandconfig
chmod +x extract-configs.sh
./extract-configs.sh
```

### 2. Read the Documentation
```bash
# View the complete setup guide
cat docs/SETUP-GUIDE.md

# View required packages
cat docs/PACKAGES.md

# View font requirements
cat fonts/FONTS.md
```

### 3. On a Fresh Arch Install

Follow these steps in order:

1. Install required packages (see `docs/PACKAGES.md`)
2. Install fonts (see `fonts/FONTS.md`)
3. Deploy configurations (see `docs/SETUP-GUIDE.md`)
4. Configure monitors and wallpapers
5. Start Hyprland

## Key Configuration Files

### Hyprland
- `hypr/hyprland.conf` - Main Hyprland configuration (sources other files)
- `hypr/UserConfigs/UserKeybinds.conf` - Your custom keybindings
- `hypr/UserConfigs/Startup_Apps.conf` - Applications launched on startup
- `hypr/monitors.conf` - Monitor configuration

### Waybar
- `waybar/configs/default-config` - Active Waybar configuration
- `waybar/style/default-style.css` - Active Waybar styling
- `waybar/Modules` - Module definitions (battery, network, etc.)

### Kitty
- `kitty/kitty.conf` - Complete Kitty terminal configuration

### Rofi
- `rofi/config.rasi` - Rofi main configuration
- `rofi/master-config.rasi` - Rofi theme configuration

### Neovim
- `nvim/init.lua` - Neovim initialization (NvChad based)
- `nvim/lua/` - Lua configuration files

## Essential Keybindings

| Keybinding | Action |
|------------|--------|
| SUPER + Return | Open Kitty terminal |
| SUPER + D | Open Rofi launcher |
| SUPER + F | Open file manager |
| SUPER + B | Open Firefox |
| SUPER + V | Clipboard manager |
| SUPER SHIFT + W | Select wallpaper |
| CTRL ALT + W | Random wallpaper |
| SUPER SHIFT + H | Show keybind hints |
| SUPER SHIFT + R | Refresh Waybar/Rofi |

## Customization

### Change Wallpaper Interval
Edit `hypr/UserScripts/WallpaperAutoChange.sh`:
```bash
INTERVAL=1800  # Change this value (in seconds)
```

### Change Default Terminal
Edit `hypr/UserConfigs/01-UserDefaults.conf`:
```bash
$term = kitty  # Change to your preferred terminal
```

### Modify Waybar Layout
Edit `waybar/configs/default-config` to adjust modules position and visibility.

### Add Custom Keybindings
Edit `hypr/UserConfigs/UserKeybinds.conf` to add your custom keybindings.

## Troubleshooting

See `docs/SETUP-GUIDE.md` for detailed troubleshooting steps.

Common issues:
- Waybar not showing → Check if running with `pgrep waybar`
- Icons missing → Install nerd fonts and run `fc-cache -fv`
- Wallpaper not changing → Check SWWW daemon and wallpaper directory
- Keybindings not working → Verify config file syntax and reload Hyprland
